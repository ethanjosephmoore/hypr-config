Ethan Moore's Hyprland config and dotfiles. Designed for Arch Linux ARM.

Sidebars scripted in yuck and CSS through EWW.
  There are additional scripts needed to run some modules, such as Python, JSON and sh files.
  The email module is connected to Thunderbird Mail via Owl for Exchange (how I receive my Outlook emails)

Kitty is the default terminal
Thunar is the default file manager
Zen is the default browser

The general gtk theme is 'Arc Dark'
